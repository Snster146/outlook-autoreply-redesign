# outlook-autoreply-redesign
Re-designed the microsoft outlooks user interface with a focus on the automatic replies page. Microsoft automatic replies page's user interface is outdated and its functionality lacks the ability to set automatic replies to occur on specific days between date ranges. 

My redesign focusses on design patterns, usability principles specifically nielsons 10 usability principles, information and visual layout and erorr prevention and detection. 

In this repository you will find an file with my interface prototype, a pdf documenting my decisions and a video explaining my re-designed interface.
